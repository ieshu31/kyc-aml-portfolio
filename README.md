# KYC/AML Compliance Portfolio

Projects built while preparing for KYC/AML and compliance analyst roles.

## Project 1 — KYC Case Tracker
Built in Google Sheets. Tracks 20 fictional customers with risk rating, 
document status, and review dates. Uses VLOOKUP, conditional formatting, 
and COUNTIF for case summary.

## Project 2 — Customer Data Validator
Python script using pandas. Reads customer data and flags sanctioned 
country exposure, missing documents, and high risk customers. Exports 
a combined exception report.
